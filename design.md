# The Baby Is Me / 赤僕 - デザインドキュメント

## アプリ概要

ユーザーが赤ちゃん役となり、AI（パパ・ママ）にあやしてもらう体験型モバイルアプリ。
癒し・リラックス・育児共感を提供する。

---

## スクリーン一覧

### 1. 言語選択画面 (LanguageSelectScreen)
- **目的**: アプリの言語を選択
- **コンテンツ**: 
  - アプリロゴ
  - 「English」ボタン
  - 「日本語」ボタン
- **レイアウト**: 中央揃え、縦並び

### 2. 保護者選択画面 (ParentSelectScreen)
- **目的**: パパ/ママを選択
- **コンテンツ**:
  - 「パパ」カード（ブルー系）
  - 「ママ」カード（ピンク系）
  - 各カードにキャラクター説明
- **レイアウト**: 2カラム横並び

### 3. メインインタラクション画面 (MainScreen)
- **目的**: 赤ちゃんアクションを選択し、AIの応答を受ける
- **コンテンツ**:
  - 上部: 選択した保護者のアバター表示
  - 中央: AI応答表示エリア（吹き出し形式）
  - 下部: 6つのアクションボタン
    - 😭 泣く / Waaah
    - 😊 笑う / Giggle
    - 😴 眠い / Sleepy
    - 🍼 お腹空いた / Hungry
    - 🎵 遊びたい / Play
    - 🤗 抱っこ / Hold me
- **レイアウト**: 
  - 上部固定エリア
  - スクロール可能な会話エリア
  - 下部固定のアクションボタングリッド（3x2）

---

## 主要コンテンツと機能

### 言語サポート
- 日本語 / 英語の2言語対応
- AsyncStorageで言語設定を保存

### 保護者キャラクター
| キャラクター | 性格 | カラー |
|-------------|------|--------|
| パパ | 優しく穏やか、少し不器用だが一生懸命 | ブルー系 (#E3F2FD, #90CAF9) |
| ママ | 温かく包容力があり、直感的 | ピンク系 (#FCE4EC, #F48FB1) |

### 赤ちゃんアクション
| アクション | 日本語 | 英語 | 絵文字 |
|----------|--------|------|--------|
| 泣く | ふえーん | Waaah | 😭 |
| 笑う | あははっ | Giggle | 😊 |
| 眠い | ねむい... | Sleepy... | 😴 |
| お腹空いた | おなかすいた | Hungry | 🍼 |
| 遊びたい | あそぼ! | Play! | 🎵 |
| 抱っこ | だっこ | Hold me | 🤗 |

### AI応答
- サーバーサイドのLLM機能を使用
- 30-80語程度の短い応答
- アクション描写を含む（*優しく抱き上げて* など）
- 保護者キャラクターと言語に応じた口調

---

## ユーザーフロー

### メインフロー
1. アプリ起動
2. 言語選択（English / 日本語）
3. 保護者選択（パパ / ママ）
4. メイン画面表示
5. アクションボタンをタップ
6. AIが応答を返す
7. 繰り返し（5-6）

### 設定変更フロー
- メイン画面から言語/保護者を変更可能（設定ボタン）

### ママモード

- Background: #FCE4EC (ペールピンク)

### 共通
- Foreground (Text): #11181C (ダーク) / #ECEDEE (ライト)

- Border: #E5E7EB / #334155

## アニメーション

- ボタン押下: scale 0.97 + haptic feedback
- AI応答: フェードイン (250ms)
- 画面遷移: スライド

---

## タブ構成

このアプリはシングルフロー設計のため、タブバーは使用しない。
代わりにスタック型ナビゲーションを使用:
- LanguageSelect → ParentSelect → Main

---

## 技術的考慮事項

- 状態管理: React Context + AsyncStorage
- AI統合: サーバーサイドLLM API
- スタイリング: NativeWind (Tailwind CSS)
- ナビゲーション: Expo Router (スタック)
